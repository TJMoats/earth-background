# earth-background
Every 5 minutes sets your desktop to the most recent image from the Himawari-8 satelite.

## Installation

Clone the repo and run:

```bash
$ npm install
```

## Simple run

```js
node app.js
```

## Example background

The background is 2200x2200 by default, but you can change that in the code.
